# S x S RAW2BMP
- *.raw 파일을 *.bmp로 만들어주는 C API
- 제작자: 강릉원주대학교 전자공학과 20171041 이민섭
- compiler: gcc

# Usage(CLI)
```shell
$ gcc main.c -o main
$ ./main
```
